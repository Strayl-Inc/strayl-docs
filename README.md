# strayl-docs

Documentation site for [Strayl](https://strayl.dev) — an AI-powered development platform.

Built with [Next.js](https://nextjs.org/), [Fumadocs](https://fumadocs.dev/), and [Tailwind CSS](https://tailwindcss.com/).

## Development

```sh
bun install
bun run dev
```

Open [http://localhost:4000](http://localhost:4000) in your browser.

## Stack

- **Framework** — Next.js 15 (App Router, static export)
- **Docs engine** — Fumadocs (MDX)
- **Styling** — Tailwind CSS v4
- **Components** — Base UI
- **Fonts** — Geist, Bitcount Prop Single, PaperMono
